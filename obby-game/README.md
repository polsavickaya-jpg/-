# Sync Obby Lab Light (Replit)

## Что это
Синхронный открытый урок в одном сайте:
1. Выбор препятствия
2. Дизайн объекта
3. Нейросети (фон игры)
4. Python-параметры препятствия
5. Общая 2D игра

## Быстрый запуск в Replit
1. Загрузите архив
2. Откройте Shell и выполните:
   - `npm install`
   - `npm start`
3. В Secrets добавьте:
   - `OPENAI_API_KEY` (необязательно)
   - `TEACHER_PIN` (необязательно, по умолчанию 1234)

## Publish (важно)
- Run command: `npm start`
- Health check path: `/health`
